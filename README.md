# VBNET-Navigation
VBNET-Navigation es una extensión para Visual Studio Code que mejora la navegación en proyectos VB.NET. Permite navegar fácilmente a través de métodos, clases, módulos y variables con funciones como "Ir a definición" (F12) y "Ir a declaración", optimizando la productividad en proyectos complejos.
